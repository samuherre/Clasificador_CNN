# 🧠 MNIST CNN Classifier - Streamlit

Este proyecto implementa un clasificador de dígitos usando una Red Neuronal Convolucional (CNN) sobre el dataset MNIST.

## 🚀 Características
- Modelo CNN con TensorFlow/Keras
- Interfaz interactiva con Streamlit
- Selección dinámica de imágenes
- Predicción en tiempo real

## ▶️ Ejecución

### 1. Instalar dependencias
```bash
pip install -r requirements.txt
```

### 2. Entrenar modelo
```bash
python model.py
```

### 3. Ejecutar app
```bash
streamlit run app.py
```
## 🧠Dataset

MNIST (28x28 imágenes de dígitos del 0 al 9)
